Welcome to my Robot car! It can be controlled from a web interface. To-Do: Add virtual joystick support for mobile devices.

Instructions on how to run PiCar

To start piCar up click "PiCar" at the top of the page. Once it has started you should see a video feed from the piCamera (not implemented yet) the controls are simple, WASD for movement. space is the brakes. Q will cause the python script to exit. There shouldn't really be a need to manually quit since reloading the page will kill the previous python script.
